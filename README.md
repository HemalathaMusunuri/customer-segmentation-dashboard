# Customer Segmentation Dashboard

## Objective
Segment customers based on demographics and purchasing behavior.

## Tools Used
- Power BI
- Excel

## Features
- Customer Segmentation
- Revenue Analysis
- City-wise Analysis
- Age Distribution
- Interactive Filters

## Insights
- High Value customers generate the highest revenue.
- Medium Value customers form the largest segment.
- Customer distribution varies across cities.
- Purchase behaviour differs among age groups.
